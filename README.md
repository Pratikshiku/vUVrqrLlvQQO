### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 搜索商品
4. 加入购物车
5. 下单商品
6. 购买历史
7. 修改密码
8. 购物卡充值
9. 安全退出

**管理员功能：**
1. 登录
2. 修改密码
3. 商品类别维护
4. 商品类别添加
5. 商品维护
6. 商品添加
7. 用户管理
8. 出售记录
9. 关于我们
10. 安全退出
